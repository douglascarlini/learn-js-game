# JS Game for Studies
HTML5 Game - Space Shooter ([demo](http://douglascarlini.com/files/github/learn-js-game/))

# About
I create this game in 2012 for improve my JavaScript skills.

# How to Play
- **N** create new ship
- **WASD** move ship
- **T** shoot
